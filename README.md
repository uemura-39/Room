# Room

自宅のリビングや自室などの写真を投稿し紹介する事が出来るアプリです。

# 制作背景
私は部屋のレイアウトを考えるのが苦手です。なので他の人がどの様なレイアウトをしているかなどを参考にしたい為、自分のレイアウトを気軽に紹介出来るサービスがあればと思い制作しました。

# 使用技術
・Ruby 2.5.1   
・Rails 5.2.3  
・MySQL


# 機能一覧
・ユーザー機能  
・ユーザー編集機能  
・ログイン/ログアウト機能  
・かんたんログイン機能  
・投稿/削除/編集機能  
・コメント機能   
・いいね/いいね取り消し機能
